
-----------------------------------------------------------[Bullet Formula]-----------------------------------------------------------

필요요소 = 사거리 , 스피드 , 높이 

사거리 = 높이가 높으면 사거리가 짧아져야한다.(스피드가 높아져도 사거리는 그대로)
-> 사거리 = 사거리 / 높이

스피드 : 높이가 높으면 스피드도 짧아진다.(사거리가 높아져도 스피드는 그대로)
-> 스피드 = 스피드 / 높이

높이 : 기본 10   10이상이면 모든 물체와 상호작용하지않고 최대 높이까지 올라갔다가 다시 떨어진다.
-> 포물선으로 이동하는 스피드 = 기본값 * 높이
